## PART 1: CREATING A LISTING

### Step 1: Photos (5-10 images)

**Upload Order:**
1. **Main Image** - Product mockup (most important!)
   - 2000×2000px recommended
   - Clean, professional
   - Show the actual product
   - White or styled background

2. **Preview Image** - Show all pages included
3. **Features Image** - List what's included
4. **Color Options** - Show variations
5. **Mobile Preview** - Show it works on phones
6. **Instructions** - Show how easy it is

**Photo Tips:**
✓ Use natural lighting
✓ Consistent style across all listings
✓ Add text overlays for key info
✓ Show scale (laptop/phone/mockup)
✗ No blurry images
✗ No screenshots as main image
✗ No watermarks covering product

---

### Step 2: Listing Details

**Title Formula:**
```
[Main Keyword] + [Style] + [Feature] + [Year] + [File Type]
```

**Example:**
```
Modern Resume Template ATS Friendly 2026 | Professional CV Template | Canva Editable | Instant Download | Job Application
```

**Title Tips:**
- 140 characters max
- Front-load important keywords
- Use pipes (|) to separate elements
- Include "2026" for freshness
- Add "Instant Download"

### Step 5: Pricing

**Price Formula:**
```
Base Price + Perceived Value - Competitor Check
```

**Recommended Pricing:**
| Product Type | Price Range |
|--------------|-------------|
| Single template | $3.99 - $5.99 |
| Bundle (3-5) | $8.99 - $12.99 |
| Mega bundle | $19.99 - $29.99 |

**Pricing Tips:**
- Start lower to get initial sales/reviews
- Raise prices after 10-20 sales
- Use psychological pricing ($4.99 vs $5)
- Offer bundle discounts

## PART 2: UPLOADING FILES

### Digital File Upload

**File Types Accepted:**
- PDF ✓
- JPG/PNG ✓
- ZIP ✓ (best for multiple files)
- TXT (for Canva links) ✓

**Best Practice - ZIP Method:**
1. Create folder with all files
2. Include README.txt with instructions
3. Zip the folder
4. Upload to Etsy
5. Max file size: 20MB per file

**ZIP Structure:**
```
Resume-Template-Modern-Navy.zip
├── Canva-Template-Link.txt
├── Resume-Template.pdf
├── Resume-Template.docx
├── Cover-Letter.pdf
└── README-Instructions.txt
```

## PART 4: PUBLISHING SCHEDULE

### Batch Upload Strategy

**Week 1:** Upload 10 resume templates
- 2 per day, 5 days
- Space out by 2-3 hours

**Week 2:** Upload 10 more
- Continue pattern

**Why Not All at Once?**
- Etsy algorithm favors active shops
- Fresh listings get temporary boost
- Easier to manage customer service
- Time to optimize based on performance

### Best Times to Publish

**Days:** Tuesday - Thursday (best)
**Times:** 
- 9-11 AM EST (US morning)
- 7-9 PM EST (US evening)
