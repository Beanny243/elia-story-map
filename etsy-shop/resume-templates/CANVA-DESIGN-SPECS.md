# CANVA DESIGN SPECIFICATIONS
## Resume Templates - Design Guide

---

## Template 1: Modern ATS-Friendly Resume

### Page 1 - Resume
**Dimensions:** 8.5 × 11 inches (Letter) or A4

**Layout:**
```
┌─────────────────────────────────────┐
│  NAME (24pt Bold)                   │
│  Title/Headline (12pt)              │
│  Contact Info Bar (9pt)             │
├─────────────────────────────────────┤
│                                     │
│  PROFESSIONAL SUMMARY               │
│  (2-3 lines, 10pt)                  │
│                                     │
│  WORK EXPERIENCE                    │
│  • Job Title - Company (11pt Bold)  │
│    Dates | Location (9pt)           │
│    - Achievement 1 (10pt)           │
│    - Achievement 2                  │
│                                     │
│  (Repeat for 2-3 jobs)              │
│                                     │
│  EDUCATION                          │
│  • Degree - School                  │
│    Graduation Date                  │
│                                     │
│  SKILLS                             │
│  Skill 1 • Skill 2 • Skill 3        │
└─────────────────────────────────────┘
```

**Colors:**
- Navy: #1E3A5F
- White: #FFFFFF
- Accent: #4A90A4
- Text: #333333

**Fonts:**
- Headers: Montserrat Bold
- Body: Open Sans Regular

## Export Settings

### For Etsy Upload:
- **PDF:** Print quality, CMYK
- **Images:** JPG, 300 DPI
- **Mockups:** JPG, 72 DPI (web)
- **Thumbnails:** 570×456px (Etsy recommended)

### File Naming Convention:
```
Resume-Template-[Style]-[Color]-[Page].pdf
Example: Resume-Template-Modern-Navy-Page1.pdf
```

---

## Quick Create Process

1. **Duplicate base template in Canva**
2. **Change color palette** (apply saved colors)
3. **Update fonts** (if needed for style)
4. **Adjust layout** (add/remove sections)
5. **Export all formats**
6. **Create mockups**
7. **Package for Etsy**

Time per template: 15-20 minutes
Time for 30 products: 7-10 hours
